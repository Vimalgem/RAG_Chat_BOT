# 🧠 Fusion RAG System

A **Fusion RAG** system built with **LangChain**, **Groq**, and **Streamlit**.

## 📦 Features

- Upload any PDF file
- Ask questions based on its content
- Uses **multi-query retrieval** and **hybrid search**
- Powered by **Llama3 via Groq API**
- Uses **open-source embeddings** and **FAISS** for retrieval
